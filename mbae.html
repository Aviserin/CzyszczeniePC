Malwarebytes Anti-Exploit

Malwarebytes Anti-Exploit - to bezpłatna aplikacja, która chroni w czasie rzeczywistym popularne programy w naszym systemie. Program może być używany równocześnie z innym programem antywirusowym, nie pobiera sygnatur, nie posiada piaskownicy. Zużywa minimalne zasoby komputera.

Malwarebytes Anti-Exploit chroni takie programy jak:

  1. Przeglądarki/komponenty:
    • Mozilla Firefox + dodatki.
    • Google Chrome + dodatki.
    • Microsoft Edge + dodatki.
    • Internet Explorer + dodatki.
    • Opera + dodatki.
    • Tor Browser.
    • Seamonkey.
    • Comodo IceDragon.
    • Java.
    • Adobe Flash Player.
    • Shockwave.
  2. Aplikacje:
    • Adobe Acrobat.
    • Adobe Reader.
    • Foxit Phantom.
    • Foxit Reader.
    • LibreOffice.
    • Pakiet Microsoft Office.
    • Windows Media Player.
    • VLC.
    • Winamp.
    • QuickTime.

Program jest przeznaczony głównie do użytku komercyjnego (biznesowego), lecz jest możliwość korzystania z wersji darmowej BETA (rozwojowej), która oferuje ograniczoną ochronę dla przeglądarek i programu Java.
