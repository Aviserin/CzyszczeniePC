TronScript

TronScript - to skrypt, który walczy dla użytkownika (ang. fight for the user). To darmowy skrypt open-source, który automatyzuje proces dezynfekcji i czyszczenia systemów Windows.
Celem Tron'a jest źle działający systemu Windows (zasobożerne programy, zainfekowany system itp.) i zautomatyzowanie około 85% działań związanych z dezynfekcją, usuwaniem zasobożernych programów i zasadniczo zoptymalizowania systemu aby działał znów jak zaraz po świeżej instalacji.

Przebieg działania skryptu Tron:

  1. Wstęp:
    • Tworzenie punktu przywracania systemu.
    • Rkill.
    • ProcessKiller.
    • Naprawa WMI.
    • McAfee Stinger.
    • TDSSKiller.
    • Usunięcie starych punków przywracania systemu.
    • Kopia zapasowa rejestru.
    • Czyszczenie VSS.
  2. Usuwanie tymczasowych plików:
    • Czyszczenie Internet Explorer.
    • Narzędzie CCleaner.
    • Narzędzie BleachBit.
    • TempFileCleanup.bat.
    • USB Device Cleanup.
    • Zduplikowane pliki.
    • Dziennik zdarzeń Windows.
    • Usługi aktualizacje Windows.
  3. Usuwanie zbędnych programów:
    • Programy OEM.
    • Toolbary/BHO.
    • Metro jeśli nie jest używany.
    • OneDrive jeśli nie jest używany.
  4. Dezynfekcja:
    • Czyszczenie pamięci CryptNet SSL.
    • Malwarebytes Anti-Malware.
    • KVRT.
    • Sophos Virus Removal Tool.
  5. Naprawa:
    • Windows Installer.
    • DISM.
    • Reset uprawnień rejestru.
    • Reset uprawnień plików.
    • System File Checker.
    • chkdsk.
    • Wyłączenie Telemetrii.
    • Wyłączenie migracji do Windows 10.
    • Sieć.
    • Rozszerzenia plików.
  6. Łatki:
    • 7-Zip.
    • Adobe Flash Player.
    • Adobe Reader.
    • Java Runtime Environment.
    • Aktualizacje Windows.
    • Reset DISM.
  7. Optymalizacja:
    • Plik wymiany (pagefile.sys).
    • Defraggler.
  8. Tworzenie logów.
  9. Możliwość dadania własnych skryptów (opcjonalnie).
  10. Ręczne narzędzia:
    • ADSSpy.
    • AdwCleaner.
    • aswMBR.
    • autoruns.
    • ComboFix.
    • PCHunter.
    • Junkware Removal Tool.
    • Net Adapter Repair.
    • ServicesRepair.
    • Tron Reset Tool.
    • VirusTotal uploader tool.

Projekt TronScript budowany jest z dużym naciskiem na wkład społeczność REDDIT i regularnie aktualizowany.
Uwaga: w zależności od stanu naszego komputera, poziomu infekcji, zapełnienia dysków oraz sprzętu, TronScript może działać nawet od 4 do 6 godzin.
