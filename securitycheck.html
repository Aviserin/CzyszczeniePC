SecurityCheck

SecurityCheck - to program skanujący popularne aplikacje zainstalowane w systemie celem sprawdzenia ich aktualności. Po zakończeniu skanowania, program SecurityCheck wyświetli plik raportu zawierający informacje o systemi jak i aktualności programów zainstalowanych na komputerze. Plik raportu jest sformatowany pod najpopularniejsze fora internetowa.

Plik raportu utworzony przez program SecurityCheck jest podzielony na 5 sekcji:

  1. Informacje o systemie operacyjnym, przeglądarce systemowej.
  2. Program antywirusowy, zapora systemu.
  3. Program antymalware, programy narzędziowe jak: Java, Adobe Reader, Flash Player.
  4. Lista uruchomionych procesów.
  5. Kondycja systemu Windows.

Program tworzy raport w postaci pliku:
checkup.txt
