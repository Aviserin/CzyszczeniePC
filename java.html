Java SE Runtime Environment

Java SE Runtime Environment - Platforma Java, Standard Edition (Java SE) pozwala na tworzenie i wdrażanie aplikacji Java na komputerach stacjonarnych i serwerach, a także w wymagających środowiskach wbudowanych. Java oferuje bogaty interfejs użytkownika, wydajność, wszechstronność, przenośność i bezpieczeństwo, których wymagają dzisiejsze aplikacje.

Ostatnie wydanie platformy Java zawiera ważne poprawki zabezpieczeń. Firma Oracle zdecydowanie zaleca wszystkim uaktualnienie do najnowszej wersji Java SE.

Java to nie to samo co JavaScript, będący prostą technologią stosowaną do tworzenia stron internetowych i działającą tylko w przeglądarce.
